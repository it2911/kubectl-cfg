> kubectl cfg is a kubectl plugin that provides you a easier way to manage the config of your Kubernetes cluster.

### 🏠 [Homepage](https://github.com/it2911/kubectl-cfg)

## Install

```sh
kubectl krew install kubectl-cfg
```

## Usage

```sh
kubectl cfg --help
```
